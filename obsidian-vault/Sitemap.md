# Sitemap

Nota espejo de [docs/INSTRUCCIONES_SITEMAP.md](../docs/INSTRUCCIONES_SITEMAP.md).

## Para que sirve

Mantiene el mapa XML del sitio actualizado para que Google descubra paginas nuevas y entienda cambios importantes.

## Cuándo revisarlo

- Al crear paginas nuevas.
- Al modificar servicios, blog o homepage.
- Antes de un deploy importante.
- Cuando Search Console no detecta una URL esperada.
