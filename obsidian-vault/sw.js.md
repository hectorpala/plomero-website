# sw.js

Ruta: `sw.js`

## Para que sirve

Service worker del sitio. Controla cache/PWA y puede afectar si usuarios ven CSS o contenido viejo.

## Riesgo

Si no se bumpea la version de cache cuando aplica, el navegador puede seguir mostrando recursos viejos.
